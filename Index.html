Numpad Binds Generator

This is a numpad bind generator for Counterstrike Global Offensive. Using the interface below you will be able to generate buy binds to be used in game. For example, a full buy in game can require up to 27 inputs, using up the freezetime you have at the beginning of the round to talk strategy. With a buy bind all of these 27 inputs can be executed with 1 button.

How to:

  • Click your key of choice from the numpad on the left,
  • Select the weapons, gear and grenades you want,
  • Copy the code from the output box below and either:
  • A) Paste the code line by line into your console, then type host_writeconfig,
  • B) Paste the entire code into your autoexec and save.
